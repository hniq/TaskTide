# 不咕了 - DDL 任务规划器

一个面向拖延症人群的智能任务管理工具，基于艾森豪威尔矩阵进行优先级管理，支持 AI 智能任务解析。

## 功能特性

### 核心功能
- **AI 智能解析**: 自然语言输入 (Ctrl+K)，自动提取任务标题、截止日期、紧急/重要度，支持自动拆分子任务
- **AI 预览确认**: AI 解析后先展示预览卡片，用户可编辑后再确认创建，避免误操作
- **艾森豪威尔矩阵**: 二维 urgency x importance 映射到 Q1-Q4 四象限，点击象限可快速筛选
- **日历视图**: 月视图/周视图，支持拖拽任务到任意日期完成分配并更新截止日期
- **智能拆分**: AI 自动拆分大任务为子任务，先展示预览结果，用户确认后才应用
- **自动分配**: 贪心算法按优先级将子任务分配到每日计划，仅对未手动分配的子任务执行

### 体验优化 (v2.1)
- **无截止日期任务支持**: 无截止日期的任务可正常自动分配（Q2/Q4分配到周末，Q1/Q3分配到两周内）
- **单任务自动分配**: 任务详情页新增"自动分配"按钮，可为单个任务执行智能分配
- **已完成任务管理**: 侧边栏已完成区域支持恢复/删除按钮，操作更便捷
- **日历删除按钮**: 月视图/周视图的任务项支持一键删除移除分配
- **多种排序**: 支持按截止日期（近→远）、按优先级（高→低）、手动排序三种模式
- **手动排序**: 选择手动排序模式后可拖动任务卡片调整顺序
- **内联编辑**: 任务详情面板支持直接编辑标题、截止日期、重要/紧急程度、备注，无需弹窗
- **拆分集成**: 智能拆分功能集成在任务详情页，支持一站式查看/编辑/拆分
- **子任务拖动排序**: 详情页中子任务支持拖动调整顺序

## 快速开始

### 本地开发

1. 安装后端依赖
```bash
cd backend
npm install
```

2. 配置环境变量
```bash
export DASHSCOPE_KEY=your_modelscope_api_key
```

3. 启动后端服务
```bash
npm run dev
```

4. 打开前端页面
```bash
# 直接打开 frontend/index.html，或使用任意静态服务器
npx serve frontend
```

### 魔搭创空间部署

1. Fork 本项目到您的仓库
2. 在魔搭创空间创建新项目，关联仓库
3. 在创空间设置中配置环境变量 `DASHSCOPE_KEY`
4. 部署即可

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+K | 聚焦 AI 输入框 |
| Ctrl+N | 新建任务 |
| Esc    | 关闭所有弹窗/面板 |

## 技术栈

- **前端**: 原生 HTML/CSS/JS (ES6+ 模块化)，零依赖
- **后端**: Node.js + Express
- **AI**: 魔搭 ModelScope API (通义千问)
- **存储**: localStorage 本地持久化
